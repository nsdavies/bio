Pokemon Game

I created a very rudimentary Pokemon Game during my 1st year at Oregon State. A 9X9 grid that a player was in and the whole goal of really any Pokemon game is to catch all the Pokemon, in this case 12, and to evolve them all to their final form. The user caught Pokemon by throwing pokeballs at them at each encounter until it was caught with each having a different catch rate. On the grid, there were at least 2 caves that were generated which would give the player pokeballs or a moonstone, which were used to evolve, at random. The most challenging part of this game was evolving the Pokemon when the player obtained a moonstoone. I enjoyed creating a dumb AI for this game to beat in very short amount of time.
Technologies used: C++, Atom

Bash shell

In my OS1 class, one of our assignments was to implement a bash shell in C. We had to manually write the functionality for cd, ls, and exit. I was also tasked with forking each child and using pthreads and mutexes for the background processes for the shell. This was by far my most difficult assignment in any class I have taken. The background process was the biggest challenge as it was very hard to figure out how to keep track of all the children and make sure they were returned and not turned into zombies while another proces was executed.
Technologies used: C, Atom

  • Class: CS 344
  • Date: Fall 2018
  • Development Type: Shell Development

Gesture Recognition

For my Senior Design Project, I was selected for the Intel Gesture Recognition project. This Intel Gesture recognition project is a Machine Learning model to classify the ASL sign language using the Intel Real sense depth camera. The goal of this project was to create an affordable solution to communicate in communities that use sign language. The machine learning model uses a ResNet 18 data set and we are using the depth in lieu of the 3 color bands. The software portion of the code, which I worked on, acts as a large pipeline of data from the user interface, camera, and the machine model. We implemented our pipeline using Python in order to utilize the library for the camera.
Technologies used: Python, Pytorch, PyQT5, Intel Realsense library
